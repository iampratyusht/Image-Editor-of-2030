DATASET DESCRIPTION:

VIDIT (Virtual Image Dataset for Illumination Transfer) is a synthetic dataset
designed for relighting and illumination transfer tasks. It contains:

- 300 virtual scenes rendered in Unreal Engine
- 8 light directions: N, S, E, W, NE, NW, SE, SW
- 5 color temperatures: 2500K, 3500K, 4500K, 5500K, 6500K
- Ground truth depth maps
- Total: 300 × 8 × 5 = 12,000 images

The Hugging Face version (Nahrawy/VIDIT-Depth-ControlNet) includes:
- scene: Scene identifier
- image: RGB image (PIL)
- depth_map: Depth map (PIL)
- direction: Light direction string (N, S, E, W, NE, NW, SE, SW)
- temperature: Color temperature
- caption: Text description
